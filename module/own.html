How to use?


ALL NEWS

https://news-api-public.herokuapp.com/news

By using Fetch API, To get all news (here all category news like sports, india news, world news etc.) from api, You need to add endpoint '/news' to 'https://news-api-public.herokuapp.com'

NOTE ! THIS API ONLY WORKS ON GET REQUEST including all endpoints


After getting response, You can see all news in json format. You can use it by looping methos to get particular data

It is Araay of objects, and each object contains one news data, In one each object there are 4 properties

  • category - type of news
  • title - title of news
  • image - image of news
  • fullStory - get the full story of news

    To get full story of particular news, take data from fullStory property throught looping, than fetch api again by appending particular fullStory value which is in encoded form (Why encoding? : to get response faster by doing encoding data, and fetch appended url again)
